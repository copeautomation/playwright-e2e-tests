# Working with Multiple Elements 

## List Iteration and Array Manipulation
### Methods Used
```js
// Use of CSS selector
.locator(".className")

// Count elements
.count()

// Get specific element by index
.nth(index)

// Locator chaining for nested elements
.locator(".parent").locator(".child")

// Get text content
.innerText()
.textContent()

// String manipulation
.trim()
.replace("$", "")

// Array methods
.map()
.filter()
.push()
```
